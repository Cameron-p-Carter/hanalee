 APP

This project is a Flight Booking System built with Flask (Python) for the backend. It allows users to register, log in, browse available flights, book seats, and view their bookings. The application uses Jinja2 templating for rendering dynamic web pages and JSON files for storing user and flight data. It features session-based authentication and a clean structure, providing a simple yet functional flight management system.
